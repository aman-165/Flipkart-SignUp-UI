# 🛍️ Flipkart Sign-Up Form UI

This is a **Flipkart-inspired sign-up form** built with pure HTML and CSS. It's a front-end practice project that replicates the layout and styling of a modern e-commerce account creation page.

## ✨ Features

- Flipkart-style clean and modern UI  
- Responsive and centered form box  
- Simple password guidance  
- Links for Sign In and Business Account  
- Minimal design for beginner projects

---

## 🚀 Technologies Used

- HTML5  
- CSS3  

No frameworks or JavaScript used — this is a lightweight, responsive HTML/CSS-only form.
